## Deep learning Api Server Demo

### 使用 Python 的 Flask 去啟動server介接Deep learning的資料，回傳JSON
### 使用 googletrans 將 VG19 的資料變成中文
---

第一步
安裝 Python3： https://www.python.org/downloads/


第二步 
安裝 Tensorflow
```
pip install tensorflow
```

第三步 
安裝 Theano
```
pip install theano
```

第四步 
安裝 Keras
```
pip install keras
```

執行 
```
python app.py
```
